Getting Started

Vue-rate-it is an extensable rating system for Vue.js 2. It includes three built-in rating comopnents for rating with stars, hearts and over 600 font awesome glyph, check out how simple it is to use:

Stars

<star-rating></star-rating>

Hearts

<heart-rating></heart-rating>

Font Awesome Thumbs-up Glyph

<fa-rating glyph="thumbs-up" active-color="#ffcd94"></fa-rating>

Setup

NPM

To use all vue-rate-it features you will need to install it via npm: 
npm install vue-rate-it

Once installed you can import all the rating components:

import Raters from 'vue-rate-it';

Or you can import them individually:

import {StarRating} from 'vue-rate-it';
import {HeartRating} from 'vue-rate-it';
import {FaRating} from 'vue-rate-it';
Important
You will need to ensure that you have added vue-loader and babel-loader to your list of modules if using webpack. If you are using browserify you will require the vueify and babelify transforms. You will also need to ensure that you have installed babel-preset-es2015 and added "presets": ["es2015"] to your .babelrc file. If you are using vue-cli or Laravel 5.3 and above this will already have been set up for you.

Registering the Rating Components

Global Registration

You can register your raters globally by doing the following after they have been imported: 
Vue.component('star-rating', StarRating);
Vue.component('heart-rating', HeartRating);
Vue.component('fa-rating', FaRating);

Component Registration

You can register your raters in the components that you want to use them in by doing the following:

import {StarRating} from 'vue-rate-it';

export default{
  components:{
    StarRating
  }
}

Using the CDN

It is recommended that you use vue-rate-it via NPM, however, each rating component does have a dist file available via unpkg, you won't however be able to create your own custom raters. To use the raters via CDN simply include the following in your webpage:

Star Rating

<link rel="stylesheet" href="https://unpkg.com/vue-rate-it/dist/star-rating.min.js">

Heart Rating

<link rel="stylesheet" href="https://unpkg.com/vue-rate-it/dist/heart-rating.min.js">

Fa Rating (Font-awesome)

<link rel="stylesheet" href="https://unpkg.com/vue-rate-it/dist/fa-rating.min.js">

Syncing Ratings between Parent and Child

The first thing you will want to do is sync your ratings between the parent and the rating component, here's an example of syncing in action:

Currently Selected: {{rating}}
Reset Rating

Using v-model for Syncing in Vue 2.2 and above

If you are using Vue 2.2 and above the simplest way to sync the rating is to use v-model:

Basic Markup

<heart-rating v-model="rating"></heart-rating>

Complete Example

<template>
  <div>
    <heart-rating v-model="rating"></heart-rating>
    <div>Currently Selected: {{rating}}</div>
    <div><a href="#" @click.prevent="rating = 0">Reset Rating</a></div>	
  </div>
</template>

<script type="text/javascript">	
import HeartRating from 'vue-rate-it';

export default{
  components: {
    HeartRating
  },
  data(){
    return {
      rating: 3
    }
  }
}
</script>

Syncing Ratings in Vue 2.1 and below

It isn't possible to use v-model on the component in Vue.js 2.1 and below, however, the following is the equivelent of the v-model example above:

Basic Markup

<heart-rating @rating-selected="rating = $event" :rating="rating"></heart-rating>

Complete Example

<template>
  <div>
    <heart-rating @rating-selected="rating = $event" :rating="rating"></heart-rating>
    <div>Currently Selected: {{rating}}</div>
    <div><a href="#" @click.prevent="rating = 0">Reset Rating</a></div>	
  </div>
</template>

<script type="text/javascript">	
import HeartRating from 'vue-rate-it';

export default{
  components: {
    HeartRating
  },
  data(){
    return {
      rating: 3
    }
  }
}
</script>
